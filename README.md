# Jujutsu Kaisen Website

## Project Overview
The Jujutsu Kaisen Website is dedicated to providing fans with all the information they need about the popular anime and manga series, Jujutsu Kaisen. This website features character profiles, episode guides, news, and user community sections.

## Features
- **Character Profiles**: Detailed information about main and supporting characters.
- **Episode Guides**: Summaries and reviews of each episode, along with air dates.
- **News Section**: Latest news and announcements related to Jujutsu Kaisen.
- **Community Forum**: A platform for fans to discuss episodes, characters, and theories.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/reuvenjohn22/jujutsu-kaisen-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd jujutsu-kaisen-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000`
